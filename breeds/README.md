# Breed Images

This folder contains curated candidate breed images for the Pet Cost & Vet Bill Calculator site.

Source policy:
- Use free-use sources only. Current files are from Wikimedia Commons, Public Domain Pictures, and Pexels.
- Do not hotlink remote images. The local files are saved under each breed folder.
- Keep `credit.json` and `source-url.txt` with each image.
- If the license is CC BY or CC BY-SA, add visible attribution near the image or on a credits page before publishing.
- Before final website use, visually confirm breed accuracy and avoid images that feel clinical, dated, cropped poorly, historical, mixed-breed, or unfriendly.
